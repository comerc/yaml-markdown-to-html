transform-yaml-markdown [![build][3]][4]
=======================

transform a folder of markdown files with yaml frontmatter to html

Installation
------------

```bash
npm i transform-yaml-markdown
```

Usage
-----

```bash
transform-yaml-markdown <source> <destination> <pathToRenderFunction>
```
